# Liry 的工具库monorepo

## 介绍

本仓库是一个`monorepo`, 使用`pnpm`管理

## 环境要求

1. `node` `>=` `24`
2. `pnpm` `>=` `10`

## 项目代号表

以下名称为本仓库**业务与库存包**的合法 npm 包名后缀（不含作用域）；新增同类包时须从中选用，**不得**自造代号。

1. `rigel`
2. `luna`
3. `astra`
4. `nova`
5. `polaris`
6. `vega`
7. `sirius`

### 包命名与作用域

- 本 monorepo 内发布的 npm `name` **一律**使用 **`@liry-k/`** 作用域。
- **业务/库存包**（上表所列代号对应的包）：`name` 必须为 **`@liry-k/<代号>`**（例如 `@liry-k/luna`），且 `<代号>` 须为上表之一。
- **工具包**（共享配置、构建 CLI 等，不对应上表代号）：仍须为 **`@liry-k/<描述性名称>`**（例如 `@liry-k/tsconfig`、`@liry-k/build`），名称不必出现在代号表中。

## 协作约定

自动化助手与贡献者的工程惯例见根目录 **`agents.md`**（文件与目录命名、导入别名、质量约束等）。
